Example run

This page shows an example run with a configuration of four players: one villagers, two mafiosi, and one detective. This configuration was chosen as it is the minimal configuration that incorporates both the extra knowledge of mafia and the special ability of the detective. It thus provides the clearest overview that also incorporates all rules of the model, albeit that the townfolk is not the majority.
